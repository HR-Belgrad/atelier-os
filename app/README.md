# Atelier OS Desktop Prototype v0.2

## Starten

Voraussetzung: Node.js 20 oder neuer.

```bash
cd app
npm install
npm run dev
```

Die App öffnet ein eigenständiges Desktop-Fenster. Über **Repository öffnen** kann ein lokaler Atelier-Ordner ausgewählt werden. In v0.2 wird der Pfad nur angezeigt; das Einlesen der Markdown-Objekte folgt in v0.3.

## Enthalten

- eigenständige Electron-App
- React/TypeScript-Oberfläche
- Navigation für Werke, Musik, Wissen, Bibliothek und Graph
- Repository-Auswahl per macOS-Dialog
- erster interaktiver Graph-Prototyp
- keine Abhängigkeit von Obsidian
