# Sprint v0.2 – Standalone Desktop Shell

## Ziel

Atelier OS startet als eigenständige Desktop-Anwendung und zeigt die geplanten Hauptbereiche einschließlich eines ersten Wissensgraphen.

## Fertig

- [x] Electron-Fenster
- [x] React- und TypeScript-Grundlage
- [x] Hauptnavigation
- [x] Dashboard-Prototyp
- [x] Repository-Auswahldialog
- [x] Graph-Prototyp
- [x] ADR zur Ablösung von Obsidian

## Nicht Bestandteil

- Markdown-Dateien einlesen oder verändern
- Git-Automatisierung
- Editor
- Suche
- KI-Assistent
- Packaging als signierte macOS-App

## Nächster Sprint

v0.3 liest echte Markdown-Objekte aus einem ausgewählten Repository und zeigt sie in der Oberfläche an.
